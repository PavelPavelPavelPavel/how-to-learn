# Научиться учиться

Проект для тех кто сомневается что сможет.

## Цели проекта:

1. Доказать неуверенным и сомневающимся что они смогут.
2. Вся информация по теме собранна в одном месте.

### Открываем, читаем, воодушевляемся, учимся.
